# DGFinance

Um site/aplicativo para controle financeiro, pessoal.
O objetivo principal desse app é a possibilidade de controlar todos os gastos pessoais. O aplicativo contará com uma ampla possibilidade de controles, sendo a principal delas a de cartões de crédito.

Este trabalho também irá fazer parte do Trabalho de Graduação - TG, do curso de Análise e Desenvolvimento de Sistemas da Faculdade de Tecnologia - Fatec de Ourinhos - SP.
